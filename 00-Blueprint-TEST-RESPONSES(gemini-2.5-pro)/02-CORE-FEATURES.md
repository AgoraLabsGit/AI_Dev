# Phase 0: Blueprint - Core Features & User Stories

**Last Updated**: January 2025

---

## 1. Core Feature Epics
> List the high-level features that are absolutely essential for the MVP (Minimum Viable Product).
>
> - **Epic 1**: User Authentication & Profile Management
> - **Epic 2**: AI-Powered Translation Practice
> - **Epic 3**: Interactive Reading Comprehension
> - **Epic 4**: Memorization & Spaced Repetition
> - **Epic 5**: Progress Tracking & Analytics
> - **Epic 6**: Conversation Practice (Future Enhancement)

---

## 2. User Stories
> Break down the epics into specific user stories. This helps clarify the exact requirements for each feature.
>
> ### **User Authentication & Profile Management**
> - **As a** user, **I want to** create an account with email and password **so that** I can access my personalized learning experience.
> - **As a** user, **I want to** log in securely **so that** my progress and achievements are protected.
> - **As a** user, **I want to** view and edit my profile information **so that** I can customize my learning experience.
> - **As a** user, **I want to** see my current learning level and XP **so that** I can track my progress.
>
> ### **AI-Powered Translation Practice**
> - **As a** learner, **I want to** practice translating Spanish sentences to English **so that** I can improve my translation skills.
> - **As a** learner, **I want to** receive immediate AI feedback on my translations **so that** I can learn from my mistakes.
> - **As a** learner, **I want to** see word-level evaluation with color coding **so that** I can understand which words I got right or wrong.
> - **As a** learner, **I want to** use hints when I'm stuck **so that** I can get help without giving up.
> - **As a** learner, **I want to** navigate between sentences **so that** I can practice at my own pace.
> - **As a** learner, **I want to** bookmark difficult sentences **so that** I can review them later.
>
> ### **Interactive Reading Comprehension**
> - **As a** learner, **I want to** read Spanish texts at my level **so that** I can improve my reading comprehension.
> - **As a** learner, **I want to** click on words to get translations **so that** I can understand unfamiliar vocabulary.
> - **As a** learner, **I want to** translate paragraphs and receive feedback **so that** I can practice reading comprehension.
> - **As a** learner, **I want to** filter reading materials by difficulty and genre **so that** I can find appropriate content.
> - **As a** learner, **I want to** track my reading time and comprehension score **so that** I can monitor my progress.
>
> ### **Memorization & Spaced Repetition**
> - **As a** learner, **I want to** review previously practiced sentences **so that** I can reinforce my learning.
> - **As a** learner, **I want to** use spaced repetition algorithms **so that** I can optimize my memorization.
> - **As a** learner, **I want to** mark words as "known" or "unknown" **so that** the system can adapt to my knowledge level.
> - **As a** learner, **I want to** see my memorization progress **so that** I can track my vocabulary growth.
>
> ### **Progress Tracking & Analytics**
> - **As a** learner, **I want to** view my overall progress dashboard **so that** I can see my learning achievements.
> - **As a** learner, **I want to** see detailed statistics for each learning mode **so that** I can identify my strengths and weaknesses.
> - **As a** learner, **I want to** track my daily streak and XP earned **so that** I can stay motivated.
> - **As a** learner, **I want to** view my achievement badges **so that** I can celebrate my milestones.
> - **As a** learner, **I want to** see my learning level progression **so that** I can understand my advancement.
>
> ### **Conversation Practice (Future Enhancement)**
> - **As a** learner, **I want to** practice conversational Spanish **so that** I can improve my speaking skills.
> - **As a** learner, **I want to** receive AI-generated conversation prompts **so that** I can practice realistic scenarios.
> - **As a** learner, **I want to** get feedback on my conversation responses **so that** I can improve my communication skills.
