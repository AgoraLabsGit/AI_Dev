# Welcome! How to Start the AI Development System

If you're new to this system or need a refresher, this guide will walk you through the first steps to get the AI Development Partner up and running.

The entire process begins with a single prompt to the AI. Your choice of prompt will depend on your starting point.

---

## The Universal "Boot-Up" Prompt

No matter which scenario you choose below, you will always start by giving the AI a command to read its initial instructions. This ensures that even a "fresh" AI instance knows exactly what to do.

**The core of your first prompt to the AI should be:**

> "Please read and follow the instructions in the file `00-Blueprint/00_START_HERE.md` to begin our project."

Now, choose one of the three scenarios below that best fits your situation.

---

### Scenario 1: You have an existing project you want to migrate.

If you have code you want to modernize or rebuild using this system, follow these steps.

1.  **Move Your Code:** In the file explorer, find the folder named `Legacy Project/`. Drag and drop your entire existing project's code into this folder.
2.  **Use the Migration Prompt:** Once your code is in place, send the following prompt to the AI:

    > **"Hello. I would like to migrate an existing project. I have placed the code in the `Legacy Project/` folder. Please read and follow the instructions in the file `00-Blueprint/00_START_HERE.md` to begin the migration and blueprinting process."**

---

### Scenario 2: You have a concept document or a clear idea.

If you've already written down your ideas, you can give the AI a head start.

1.  **Add Your Concept:** In the file explorer, navigate to the `00-Blueprint/` directory. Open the most relevant file (e.g., `01-PROJECT-VISION.md` or `02-CORE-FEATURES.md`) and paste your notes inside.
2.  **Use the Concept Prompt:** Send the following prompt to the AI:

    > **"Hello. I have already added my initial ideas to the documents inside the `00-Blueprint/` directory. Please read and follow the instructions in `00-Blueprint/00_START_HERE.md`, using the information I've already provided to guide our blueprinting session."**

---

### Scenario 3: You are starting a completely fresh project.

If you're starting from a blank slate with just an idea in your head, the process is very straightforward.

1.  **Use the Fresh Start Prompt:** Send the following prompt to the AI:

    > **"Hello. I would like to start a new project from scratch. Please read and follow the instructions in the file `00-Blueprint/00_START_HERE.md` to begin."**

---

After you send your chosen prompt, the AI will take over and guide you through the collaborative "Project Blueprint" phase. Just follow its lead!
